# 错误分级（L1 / L2）

| 级别 | 名称 | 处理 | 典型例子 |
|------|------|------|----------|
| **L1** | 本地可解决 | 自行修复 | lint / typecheck / 单测失败、依赖未装、本机环境 |
| **L2** | 需要升级 | 横向协调 → 上报用户 / Reviewer → 阻塞时暂停 | 契约矛盾、架构层改动、Plan 未批准、P0 门禁被迫绕过、Schema 不兼容 |

**L2 升级路径**：

1. 先横向协调（找相关 agent）
2. 无法解决 → 上报用户或 Reviewer
3. 阻塞工作流时 → **暂停当前任务**，不要擅自继续

完整定义见 [`common.md`](./common.md) §2。
